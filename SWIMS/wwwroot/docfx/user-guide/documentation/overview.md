# About This Documentation

## What Is This Site?

This is the official SWIMS documentation site, built with **DocFX** and served directly by the SWIMS application at `/docs/`. It is available to all staff who can access SWIMS — no separate login is required.

## Sections

The documentation is divided into three areas, accessible from the top navigation bar:

| Section | Who it's for |
|---------|-------------|
| **User Guide** | Day-to-day staff using SWIMS |
| **Developer Guide** | Developers maintaining or extending SWIMS |
| **API Reference** | Auto-generated C# class and method documentation |

You are currently reading the **User Guide**.

## Navigating the User Guide

Use the **left sidebar** to browse topics. The sidebar is organised by module — click any heading to expand it and see the pages within.

Use the **search bar** at the top of the page to search across all documentation. Type any keyword (e.g. "approval", "case status", "password") to find relevant pages immediately.

## Printing or Saving a Page

To print or save a documentation page as a PDF:

1. Navigate to the page you want.
2. Use your browser's **Print** function (`Ctrl+P` / `Cmd+P`).
3. Choose **Save as PDF** as the destination.

## Reporting an Issue With the Documentation

If you find information that is incorrect, outdated, or missing, contact your SWIMS administrator or the system's technical team and let them know which page and what needs correcting.

> This documentation is generated from the SWIMS source code and maintained by the development team. It reflects the version of SWIMS currently deployed in your environment.
