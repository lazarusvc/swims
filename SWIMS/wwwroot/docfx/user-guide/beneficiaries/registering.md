# Registering a Beneficiary

## Before You Begin — Search First

Always **search for an existing record** before registering a new beneficiary. Duplicate records create problems for case management and reporting.

1. Navigate to **Beneficiaries** in the sidebar.
2. Use the search bar at the top of the list to search by name or ID number.
3. If a matching record exists, use it — do not create a new one.
4. Only proceed with registration if no matching record is found.

## Registering a New Beneficiary

1. Navigate to **Beneficiaries** in the sidebar.
2. Click **Register New Beneficiary** (top-right button).
3. Complete the registration form:

### Personal Information

| Field | Required | Notes |
|-------|----------|-------|
| First Name | ✅ | |
| Middle Name | ❌ | Include if known |
| Last Name | ✅ | |
| Date of Birth | ✅ | Use DD/MM/YYYY format |
| Gender | ✅ | Select from dropdown |
| Marital Status | ❌ | Select from dropdown |

### Identity Document

| Field | Required | Notes |
|-------|----------|-------|
| ID Type | ✅ | National ID, Passport, Birth Certificate, etc. |
| ID Number | ✅ | As it appears on the document |

### Contact Information

| Field | Required | Notes |
|-------|----------|-------|
| Telephone | ❌ | Primary contact number |
| Email | ❌ | |
| Address | ❌ | Street address |
| City / District | ❌ | Select from the reference list |

4. Double-check all entries for accuracy — particularly the **ID number** and **date of birth**.
5. Click **Save**.

The beneficiary record is created and assigned a unique system ID. You can now link this record to cases and form submissions.

## After Registration

Once registered, you can immediately:

- [Create a Case](../cases/creating.md) for this beneficiary
- Link existing form submissions to the new record
- View the beneficiary's full profile
