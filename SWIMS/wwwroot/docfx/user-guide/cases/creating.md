# Creating a Case

## Before You Begin

Before creating a case, ensure:

- The **beneficiary is already registered** in SWIMS. If not, register them first — see [Registering a Beneficiary](../beneficiaries/registering.md).
- You know which **programme** the case is for.
- You have the `Cases.Manage` permission.

## Steps to Create a Case

1. Navigate to **Cases** in the sidebar.
2. Click **New Case** (top-right button).
3. Complete the case creation form:

   | Field | Description |
   |-------|-------------|
   | **Beneficiary** | Search for and select the beneficiary. Type at least 3 characters of their name or ID number. |
   | **Programme** | Select the social welfare programme this case relates to. |
   | **Notes** *(optional)* | Any initial notes about the case. |

4. Click **Save**.

The case is created with status **Pending** and assigned a case number (e.g. `CASE-2025-00042`).

## Next Steps After Creating

A new case in **Pending** status needs a **Primary Application** linked to it before the status can change:

1. Ensure the beneficiary has submitted (or you have submitted on their behalf) the relevant application form.
2. From the Case Detail page, click **Link Form**.
3. Search for the form submission and select it.
4. Check **Mark as Primary Application**.
5. Click **Link**.

Once the Primary Application is linked and approved through the approval workflow, the case status will automatically update to **Active**.

## Assigning a Case Worker

Optionally assign a case worker immediately after creating the case:

1. From the Case Detail page, click **Assign**.
2. Search for and select the staff member.
3. Click **Save Assignment**.

Assigned cases appear in the case worker's **My Cases** view.
