# Messenger — Overview

The SWIMS Messenger allows you to send and receive real-time messages with colleagues directly within the application. It is a secure, internal channel — messages stay within the SWIMS system.

## Accessing the Messenger

Click **Messenger** in the sidebar, or click the chat icon (if shown in the top bar). This opens the Messenger page.

The left panel shows your **conversation list**. The right panel shows the messages in the selected conversation.

## Conversation List

Each conversation in your list shows:

- The other participant's name
- The last message preview
- The time of the last message
- An unread count badge (if there are messages you haven't read)

Conversations are sorted by most recent activity.

## Online Status

A green dot next to a user's name indicates they are **currently online** in SWIMS. A grey dot means they are offline. Note that online status resets if the server restarts.
